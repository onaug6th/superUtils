#   常用的正则表达式

##  数字相关

整数
```js
/^[0-9]\d*$/
```

正整数
```js
/^[1-9]\d*$/
```

##  信息

邮箱
```js
/\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*/
```

身份证
```js
/^(^[1-9]\d{7}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}$)|(^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])((\d{4})|\d{3}[Xx])$)$/
```
港澳通行证验证
```js
/^[HMhm]{1}([0-9]{10}|[0-9]{8})$/;
```

台胞证验证
```js
var tbidcrad = "/(^\d{8})$/";
```

回乡证
```js
var hxidcrad = "/(H|M)(\d{10})$/";
```

台湾通行证
```js
/^[a-zA-Z][0-9]{9}$/";
```

护照验证
```js
/^1[45][0-9]{7}$|([P|p|S|s]\d{7}$)|([S|s|G|g]\d{8}$)|([Gg|Tt|Ss|Ll|Qq|Dd|Aa|Ff]\d{8}$)|([H|h|M|m]\d{8,10})$/;
```
